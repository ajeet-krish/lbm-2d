Simulation Dashboard

This interactive dashboard presents the von Kármán vortex street at multiple Reynolds numbers. Select a Reynolds number to view the velocity field, drag/lift coefficient time history, and compare against published experimental data from Williamson 1988 and Tritton 1959.

Run python3 scripts/postprocess.py output/re100 --json to generate data

Velocity Field

Velocity magnitude contour with jet colormap. The cylinder is located at x = NX/4, y = NY/2. Flow moves left to right.

Force Coefficients

Drag coefficient Cd and lift coefficient Cl time history. The drag oscillates at twice the shedding frequency; the lift oscillates at the shedding frequency. A steady case (Re ≤ 40) shows constant Cd and zero Cl.

How to generate live data. Run the simulation for each Reynolds number: ./build/LBM_Engine 100 Then convert VTK frames to JSON for the web viewer: python3 scripts/postprocess.py output/re100 --json --every 5 The JSON output goes to docs/assets/data/sim_re100.json and will be loaded by this page automatically.
