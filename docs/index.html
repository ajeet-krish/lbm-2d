The Lattice Boltzmann Method (LBM) offers an alternative to traditional Navier-Stokes discretization by tracking particle probability distributions on a regular Cartesian grid. At each node, 9 discrete distributions fi evolve through alternating collision (relaxation toward local equilibrium) and streaming (propagation to neighboring sites). The macroscopic Navier-Stokes equations emerge in the low Mach number limit via the Chapman-Enskog expansion.

This implementation uses a flat 1D memory layout for cache-optimized access, OpenMP parallel collision and streaming loops with collapse(2), Zou/He velocity inlet and convective outlet boundary conditions, and momentum-exchange force extraction for computing drag and lift coefficients on immersed obstacles. The solver is validated against the Williamson 1988 cylinder wake benchmark across a Reynolds number range of 20 to 200.

Key Results

  • D2Q9 BGK solver simulates unsteady laminar flow past a circular cylinder at Re = 20-200, resolving the von Kármán vortex street with periodic shedding at Re ≥ 60
  • Strouhal validation matches Williamson 1988 to within 5% across the laminar shedding regime (Re = 60-200)
  • Drag coefficient agrees with Tritton 1959 experimental data: Cd = 2.0 at Re = 20 (steady), Cd = 1.3 at Re = 200 (periodic shedding)
  • Cache-optimized memory layout uses flat 1D arrays (std::vector) instead of nested vectors, improving memory bandwidth utilization by eliminating pointer indirection
  • Production-grade with 14 Google Test unit tests, GitHub Actions CI on Linux and macOS, and zero compiler warnings at -Wall -Wextra
