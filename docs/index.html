LBM-2D: High-Performance C++ Lattice Boltzmann CFD Solver

A cache-optimized D2Q9 Lattice Boltzmann Method solver for 2D fluid flow around obstacles. Multi-Relaxation Time (MRT) collision, Bouzidi interpolated bounce-back for curved boundaries, and direct JSON output pipeline. Validated against canonical CFD benchmarks across eight simulation cases.

View Source on GitHub →

Why Build Another CFD Solver?

Existing open-source CFD tools like SU2 and OpenFOAM are production-grade solvers backed by decades of development. They excel at industrial-scale RANS/LES on unstructured meshes. However, they come with steep learning curves, complex build systems, and opaque codebases that obscure the core physics from the engineer.

LBM-2D takes the opposite approach: a from-scratch, readable implementation of the Lattice Boltzmann Method that prioritises clarity, cache efficiency, and rapid iteration. Every line of the solver kernel is visible in a single header. The flat 1D memory model and OpenMP-collapse(2) parallelisation are documented with their performance rationale. The Bouzidi interpolated bounce-back and MRT collision operator match modern research-grade LBM codes.

This is not a replacement for SU2 or OpenFOAM. It is a portfolio of CFD competency: demonstrating that I can implement modern LBM schemes from first principles, validate them against experimental benchmarks, and communicate the results through interactive web visualisation -- all in a production-grade C++20 codebase with CI and unit tests.

Simulation Cases

Eight validation cases spanning external aerodynamics, internal flows, urban microclimate, and automotive aerodynamics. Each case includes a comparison-slider field viewer, quantitative validation against experimental and numerical benchmarks, and discussion of accuracy.

Von Kármán vortex street across Re = 20–200. Validated against Williamson 1988 (Strouhal) and Tritton 1959 (drag coefficient). Bouzidi interpolated bounce-back for smooth curved boundary representation.

Key Features

  • MRT collision operator (default) with independently tuned moment relaxation rates -- d'Humieres 2002 formulation
  • Bouzidi interpolated bounce-back for smooth curved boundaries -- works for both circles and arbitrary polygons
  • Flat 1D memory layout -- single std::vector, cache-optimised for M1/M2/Apple Silicon prefetcher
  • OpenMP parallel collision and streaming with collapse(2) -- ~8× speedup on 8 cores
  • Momentum exchange force extraction -- exact Cd/Cl without body-fitted meshes
  • Direct JSON output per-frame velocity fields + append-only force history
  • Polygon obstacle support via point-in-polygon -- any closed 2D shape
  • 13 Google Test unit tests + GitHub Actions CI on ubuntu and macos
